# 主程序集成代码片段

下面代码片段用于说明调用关系，不要求原样复制。实际工程里请替换成你的 NUS 变量名、连接句柄和发送函数。

## 1. NUS RX 中解析命令

```c
#include "baby_badge_protocol.h"

static bool monitoring;

static void nus_rx_handler(uint8_t const *data, uint16_t len)
{
    switch (bbp_parse_command(data, len)) {
        case BBP_CMD_START:
            monitoring = true;
            break;

        case BBP_CMD_STOP:
            monitoring = false;
            break;

        default:
            break;
    }
}
```

## 2. 封装发送完整帧

```c
static bool nus_send_frame(char const *frame, size_t len)
{
    size_t offset = 0;

    while (offset < len) {
        uint16_t chunk_len = (uint16_t)(len - offset);
        if (chunk_len > 20) {
            chunk_len = 20;
        }

        ret_code_t err = ble_nus_data_send(
            &m_nus,
            (uint8_t *)&frame[offset],
            &chunk_len,
            m_conn_handle
        );

        if (err != NRF_SUCCESS) {
            return false;
        }

        offset += chunk_len;
    }

    return true;
}
```

这个简单版本只适合低速测试。正式版本请参考 `examples/nrf5_sdk_s140_nus_hooks.c`，加入 TX 队列并处理 `NRF_ERROR_RESOURCES`、`NRF_ERROR_BUSY` 和 `BLE_GATTS_EVT_HVN_TX_COMPLETE`。

## 3. 每秒发送 sample

```c
static char frame[BBP_MIN_FRAME_BUFFER_SIZE];
static uint32_t seq;
static uint32_t uptime_ms;

static void one_second_task(void)
{
    if (!monitoring) {
        return;
    }

    uptime_ms += 1000;

    size_t len = bbp_build_sample_frame(
        frame,
        sizeof(frame),
        ++seq,
        uptime_ms,
        3650,
        582,
        "supine",
        87,
        "monitoring",
        ""
    );

    if (len) {
        nus_send_frame(frame, len);
    }
}
```

## 4. 发送重复事件

```c
static uint32_t event_id;

static void report_low_battery(uint8_t battery_pct)
{
    char value[8];
    snprintf(value, sizeof(value), "%u%%", (unsigned int)battery_pct);

    uint32_t id = ++event_id;
    for (uint8_t i = 0; i < 3; ++i) {
        size_t len = bbp_build_event_frame(
            frame,
            sizeof(frame),
            ++seq,
            id,
            uptime_ms,
            "low_battery",
            "warning",
            value
        );

        if (len) {
            nus_send_frame(frame, len);
        }
    }
}
```

## 5. 接入真实传感器

```c
int16_t temperature_centi = sht45_temperature_centi();
uint16_t humidity_tenth = sht45_humidity_tenth();
const char *posture = jy61p_posture_enum();
uint8_t battery_pct = battery_percent();
```

把这些值传给 `bbp_build_sample_frame(...)` 即可，不要改协议字段名。
